How do we use data

We use the data collected to:

  • In-game User profile
    We need your User ID to link you to your in-game profile. Your ingame profile will store:
    • The coins that you own.
    • The amount of fertilizer that you own.
    • The amount of mega fertilzers that you own.
  • Server Tree
    If you own a server tree (By creating one with /start), we will store your server ID among the following ingame data:
    • Server ID: The server ID of the server where you created your server tree
    • Channel ID: the channel where treebot will post the server tree message
    • Message ID: the ID of the server tree message, the one that TreeBot will edit and update
    • webhookWebhook URL: The webhook URL for the notifications of the server tree. Tree bot will post there only when a user performs an action on the server tree
    • Tree metres: How many metres the server tree has
    • Experience: The experience of the server tree
    • Background: The background of the server tree
    • Boost: Percentage of boost (fertilizer)
    • Language: The language the tree is in (if set)
    • Tree name: How you renamed the server tree (if set)
    • Backgrounds: The backgrounds that you and your users bought for the tree
    • Role Permissions: The Role ID that you allow/deny to interact with the server tree (if set)
  • Per-Server statistics
    We link your user ID to the server ID where you interacted with the server tree in order to show you your statistics for that server as well as some other important data.
    • Server ID: The server the stat is for
    • Timestamps: Used for cooldowns
    • User ID: Your user ID
    • Water: How many times you watered the server tree
    • Fertilizer: How many times you fertilized the server tree
    • Mega Fertilizer: How many times you mega fertilized the server tree
    • Fruits: The amount of fruits you got from the tree
    • Coins: The amount of coins you got from the tree
  • Logging
    We log all your actions in the server tree, such as when you watered it, fertilized it, etc. In order to keep the game leveled and in case of any issue with the tree, we need to know what happened. Even for giving out rewards in case it affects any user profile. We also use this data to track users to see if they are abusing any exploits.
