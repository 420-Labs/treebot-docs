Privacy Policy

At TreeBot, we take your privacy seriously. This Privacy Policy outlines how we collect, use, and safeguard your information when you use our Discord Application.

When do we collect data?

We will never store any data unless the user performs an action in the bot. This includes the following:

  • Doing /shop
  • Buying a tree background in the /shop
  • Watering a tree (by using /water command or clicking the 游눥 button on a server tree)
  • Fertilizing a tree (by using /fertilize command or clicking the 游꺔 button on a server tree)
  • Mega Fertilizing a tree (by using /mega_fertilize command or clicking the 游뿺 button on a server tree)

By doing any of these actions, except the first one, you will also be linked to the server tree to show you per server statistics

How do we use data?

We use the data collected to:

  • In-game User profile
    We need your User ID to link you to your in-game profile. Your in-game profile will store:
    • The coins that you own.
    • The amount of fertilizer that you own.
    • The number of mega fertilizers that you own.
  • Server Tree
    If you own a server tree (By creating one with /start), we will store your server ID among the following in-game data:
    • Server ID: The server ID of the server where you created your server tree
    • Channel ID: the channel where treebot will post the server tree message
    • Message ID: the ID of the server tree message, the one that TreeBot will edit and update
    • webhookWebhook URL: The webhook URL for the notifications of the server tree. Tree bot will post there only when a user performs an action on the server tree
    • Tree metres: How many metres the server tree has
    • Experience: The experience of the server tree
    • Background: The background of the server tree
    • Boost: Percentage of boost (fertilizer)
    • Language: The language the tree is in (if set)
    • Tree name: How you renamed the server tree (if set)
    • Backgrounds: The backgrounds that you and your users bought for the tree
    • Role Permissions: The Role ID that you allow/deny to interact with the server tree (if set)
  • Per-Server statistics
    We link your user ID to the server ID where you interacted with the server tree to show you your statistics for that server as well as some other important data.
    • Server ID: The server the stat is for
    • Timestamps: Used for cooldowns
    • User ID: Your user ID
    • Water: How many times you watered the server tree
    • Fertilizer: How many times you fertilized the server tree
    • Mega Fertilizer: How many times you mega fertilized the server tree
    • Fruits: The number of fruits you got from the tree
    • Coins: The number of coins you got from the tree
  • Logging
    We log all your actions in the server tree, such as when you watered it, fertilized it, etc. To keep the game levelled and in case of any issue with the tree, we need to know what happened. Even for giving out rewards, in case it affects any user profile. We also use this data to track users to see if they are abusing any exploits.
