Detailed Changelog

Phew... This update took a long time, and a lot of work, but it's finally here. We hope you enjoy it!

Name Improvements

Since we implemented Tree names, we had to make some changes in how they are managed due to new requirements.

Before, you could have a tree with no name, which was nice at some point, but with this update we had to force trees to always have a name. Don't worry, this doesn't mean you need to think of a creative name for your tree right now. If your tree doesn't have a custom name, it will just use the default name (server name's Tree or the one in your language, it will auto-update as soon as you change the server tree's language)

This is a gradual process, meaning that the tree name will appear as soon as you interact with it (water, fertilize, etc.).

Design Changes

We have updated the font across the entire bot to a more modern and readable style. Additionally, we have made design updates to the shop, making it more spacious. Furthermore, purchased backgrounds in the /profile command now show only the background itself without the background name.

Removed Code

As you probably know, after some updates that changed features and commands, there was some code that was not that useful or efficient, so we did a little of a clean-up and removed it.

WebP Images

Treebot was using PNG images to display trees and shops. We decided to switch to WebP images now that the bot is verified, meaning that it can join more than 100 servers. If we continued using PNG images, the bot would be, at some point, overloading our servers due to the size of the images. Now, it's using webp images, which are smaller, and it's much faster too! Around 80% lighter! Pretty nice, right?

Badge Vault Added

This is a brand-new feature where users of your server can contribute with money to upgrade the tree badge!

  • Tree with Badge
  • /badge_vault Command Menu

Of course, Badge Vault was added as well to the /profile command, where you can see how much you or other users have contributed to the badge vault. A new button for quick access to the vault was added to the tree message.

Contributing also triggers a notification on the channel you have set as your notification channel. This contains who contributed to the badge vault and how much the contribution was.

OOP Programming

While this change will not affect you directly, TreeBot is now using OOP programming, which allows us to keep our code clean and maintainable. This also allows us to add new features with less re-coding, and faster updates.

Profile User Command

Tired of having to write /profile @user to see a user's profile? We know you are not, anyways, we added an option when right-clicking a user, so you can see its profile right away!

Leaderboards

Probably the most requested feature so far, well, we added 15 leaderboards!

You can now use the /leaderboard command to see the following leaderboards for the trees of all servers: 🌳 Metres 🌳 Times Watered 🌳 Times Fertilized 🌳 Times Mega Fertilized 🌳 Fruits Produced 🌳 Backgrounds Owned 🌳 Badge Vault

Also, the leaderboards for users around the trees: 👤 Coins 👤 Fertilizers 👤 Mega Fertilizers 👤 Times Watered 👤 Times Fertilized 👤 Times Mega Fertilized 👤 Fruits Collected 👤 Badge Vault Contribution

As we changed the way tree names are displayed and how usernames are managed, you might see '???' as a tree name at first, don't worry, the leaderboards will start displaying names properly after trees start interacting

French added!

Thanks to PSY, we now have the bot fully translated into French! You can change it for your server in the /server_lang command. Please note that Discord client commands (such as slash command autocomplete) and ephemeral messages will always display in your Discord language settings.

App Emojis

When I started coding TreeBot, for emojis we had to add them to a private server and add the bot to that server. Some time ago, discord implemented per app emojis, which allow apps to upload up to 2,000 emojis, against the 50 of a private server. We are now using app emojis!

Bug Fixes

This doesn't mean that there will be no bug at all, so if you find any, please report it to the support server, I'll do my best to fix it as soon as possible.

Customization shop completely redesigned

With this update, you can now view four backgrounds per a page, making it easier to browse the shop and find exactly what you're looking for. This enhancement reduces the total number of background pages from 36 to just 9, streamlining your shopping experience.

  • Old Shop (0.1.x)
  • New Shop (0.2.0)

As well, you can see below every background the actual price, or if the background was already bought by you or another user, it will say Bought by [user] (User can also be replaced with TreeBot for the default background or TreeBot Staff if it was given through administration commands, because of a bug or a giveaway, why not).

User Profiles

Did you ever want to see your inventory without opening the shop? It is now possible with /profile! You can see your own profile by leaving the user field empty or fill it with a user to see the profile of that user. And as we know you want to show those amazing six coins you got from a fruit, you can set the public field to Yes so other users can see your amazing progress on the server and globally.

Wait... Server progress? Yes, Since the 0.1.0, Treebot was saving how many times you watered the tree, how many times you fertilized the tree, how many times you mega fertilized the tree, how many fruits you got, and the background you have bought. Well, it is now possible to see with /profile

Fruits rates

Fruit Rates have been implemented, now you will get a certain number of fruits depending on how long the tree is. Before this was always a certain possibility to get one fruit. Now you have that possibility to get more fruit depending on the height.

Note: Fertilizers will make the fruits have more probabilities to appear, but do not give you more fruits.

Per-server cooldown

Have you ever tried to water two different servers' trees at the same time and got a cooldown message? Now you can water as many server trees as you want at the same time! This change does not modify the cooldown time, it just separates your cooldown time from the other servers.

Administration systems got better

This is not an update that users can see directly. However, with this we (TreeBot Staff) can manage user/server profiles way easier, making it better for everyone as staff will be able to solve any problem user might have.

Fertilizer now lasts longer

Before, the fertilizer and the mega fertilizer would only raise the tree boost for the next watering and nothing more. To make it more considerate with other users, the boost of the tree will get reduced to 10% by each watering, so, if my tree has a 30 % boost, it will last 3 watering 30% 20% 10% 0%

Fertilizer growth rate changed

Fertilizers had a fixed metres grow value, now it depends on the boost. For example, if the boost is 40%, the tree will grow by 4 + (40–30) / 10 * 3 = 7 metres. 4 (boost-30)/10*3)

Fruit earning is now public

When you watered a tree and got a fruit, nobody could see that amazing fruit you got, so now everyone can see what you got on the watering message! (Tree message, or the notification channel)

  • Tree Message
  • Notification channel
